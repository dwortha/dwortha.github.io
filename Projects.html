Total Property Management

Description:

This is a project that I have been working on for my father. The website is to be a small business site for a home remodeling company in Dayton Ohio. I have not completed this project due to lack of communication on design, but I left the design up to my father. Without information I was not able to complete the site, but it is still a work in progress.

Challenges:

This project was a difficult project being my first site for a company. Outside of tutorial and small static sites I had not created a site before this. I relied on resources such as w3schools to help me add functionality outside of my curriculum at Bowling Green State University. After many tutorials and YouTube videos I was finally able to fill in these gaps and start building.

First Portfolio

Description:

In this project I set out to build a portfolio to showcase my skills. I thought it would be a good idea to create a personal website where I show my skills through the site itself while also showing other projects I’ve created. This was another tutorial grind for design and functionality. I based my design off a site I have seen before and completely transformed it into a site for myself. I had a lot of fun with this site being able to do choose any lane of creativity, but I realized I could go further and created the site we are on now. Also I disabled the contact form because I didn’t want this site to connect to my email.

Challenges:

This was a challenging project because there was a lot of freedom with design and functionality which drove me to want to include design and functionality that I was not capable of implementing at the time. I overcame this by looking at different sites for a foundation and even seeing have different template were design. After all the research it was time to apply my HTML, CSS and JS skills.
